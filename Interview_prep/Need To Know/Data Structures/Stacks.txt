---=== STACKS ===---

A Stack of data.
It uses last in first out ( LIFO ) ordering

Operations:

    1. pop() - removes the first item from the Stack.
    2. push( x ) - moves x to the top of the Stack.
    3. peek() - returns the top of the Stack.
    4. isEmpty() - returns True if the Stack is empty

Stack are used a lot in recursive algorithms.
